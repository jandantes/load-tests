# load-tests

#### Development
------------------
Prerequisites
- Node.JS
- npm

Setup
- create `.env` from `.env.dist`
- for `loadtest` options, see: `https://www.npmjs.com/package/loadtest`

```
$ npm i                    # install dependencies
$ npm run test             # starts the load testing
```
